# Datos del Proyecto

Esta carpeta debe contener el archivo `Parking_Transactions.csv`.

## Descarga

Descarga el archivo CSV desde la secci√≥n de Releases del repositorio:

https://github.com/ArianaGonzabay/park-app/releases

## Instrucciones

1. Descarga `Parking_Transactions.csv` desde Releases
2. Coloca el archivo en esta carpeta (`archive/`)
3. Ejecuta el script de importacion:

```bash
npm run import:csv
```

4. Inicia el proyecto:

```bash
npm run dev
```
